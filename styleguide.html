Utility Classes

Spacing (Margin & Padding)

Utility per gestire margini e padding con scalabilità.

Scale disponibili: xs, sm, md, lg, xl, 2xl

Varianti: m, mx, my, mt, mr, mb, ml, p, px, py, pt, pr, pb, pl

.mb-md - Margin bottom medio

.mb-lg - Margin bottom largo

.p-xl - Padding extra large

<!-- Margin utilities -->
<div class="m-md">...</div>          <!-- margin all -->
<div class="mx-lg">...</div>         <!-- margin x-axis -->
<div class="mt-xl">...</div>         <!-- margin-top -->

<!-- Padding utilities -->
<div class="p-sm">...</div>          <!-- padding all -->
<div class="px-md">...</div>         <!-- padding x-axis -->
<div class="py-lg">...</div>         <!-- padding y-axis -->

Sizing

Utility per larghezza e altezza.

Width: w-auto, w-full, w-screen, w-1/2, w-1/3, w-2/3, w-1/4, w-3/4, w-1/5

Height: h-auto, h-full, h-screen

.w-full - Larghezza 100%

.w-1/2 - Larghezza 50%

<div class="w-full">...</div>       <!-- 100% -->
<div class="w-1/2">...</div>        <!-- 50% -->
<div class="w-1/3">...</div>        <!-- 33.33% -->
<div class="w-1/4">...</div>        <!-- 25% -->

<div class="h-full">...</div>       <!-- 100% height -->
<div class="h-screen">...</div>     <!-- Full viewport height -->

Display & Flexbox

Utility per controllare display e allineamento.

Display: block, inline-block, inline, flex, inline-flex, grid, hidden

Flex Direction: flex-row, flex-col, flex-row-reverse, flex-col-reverse

Align Items: items-start, items-end, items-center, items-baseline, items-stretch

Justify Content: justify-start, justify-end, justify-center, justify-between, justify-around, justify-evenly

<!-- Display -->
<div class="flex">...</div>               <!-- display: flex -->
<div class="grid">...</div>              <!-- display: grid -->
<div class="hidden">...</div>            <!-- display: none -->

<!-- Flex Direction -->
<div class="flex flex-row">...</div>     <!-- row (default) -->
<div class="flex flex-col">...</div>     <!-- column -->

<!-- Alignment -->
<div class="flex items-center">...</div> <!-- vertical center -->
<div class="flex justify-center">...</div> <!-- horizontal center -->

Responsive Classes

Le utility sono disponibili in varianti responsive con prefissi breakpoint.

Breakpoints: sm: (640px), md: (768px), lg: (1024px)

Esempio: .p-md (mobile), .md:p-lg (tablet), .lg:p-xl (desktop)

<!-- Responsive Width -->
<div class="w-full md:w-1/2 lg:w-1/3">...</div>

<!-- Responsive Grid -->
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-4">
    ...
</div>

<!-- Responsive Spacing -->
<div class="p-md md:p-lg lg:p-xl">...</div>

<!-- Responsive Display -->
<div class="hidden md:flex">Visibile solo da tablet</div>
